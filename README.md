## Установка

Для начала работы с проектом выполните следующие шаги:

1. Клонируйте репозиторий:
    ```sh
    git clone https://github.com/IluF1/bankiRu_test.git
    cd my-app
    ```

2. Установите зависимости с помощью Yarn:
    ```sh
    yarn
    ```
